How to increase Virtual Ram on Your windows PC

Creation Gadgets Tech Windows
August 6, 2020August 6, 2020Steve T IncLeave a Comment on How to increase Virtual Ram on Your windows PC

First of all let us define virtual ram :

Virtual memory is a feature of an operating system that enables a computer to be able to compensate shortages of physical memory by transferring pages of data from random access memory to disk storage.

This process is done temporarily and is designed to work as a combination of RAM and space on the hard disk.

This means that when RAM runs low, virtual memory can move data from it to a space called a paging file. This process allows for RAM to be freed up so that a computer can complete the task.

Occasionally a user might be shown a message that says the virtual memory is running low, this means that either more RAM needs to be added, or the size of the paging file needs to be increased. click here to view source

so we have defined what virtual RAM is lets learn how to increase it in windows.

How to Increase Virtual RAM in windows PC

  1. Go to this PC

    Go on the windows search panel and type in “This PC” if you dont have the short cut on your Desktop.and click it

  2. After you have opened “This PC”,right click on your mouse(a box with information will popup) and click “Properties”

  3. Click on advance settings

    When you click on “properties” you will be taken to a page that show you detailed information about your PC(e.g. memory,RAM,Name and more) so click of the advanced system settings option to your upper left

  4. Another popup will appears

    After the popup appears click on settings in the performance tab

  5. Go to advanced again

    After clicking settings another popup will appear so go to the Advanced option. You will see the selected(Adjust the best performance of_Programs_. Just go and click change

  6. Select the selected box

    They will be a selected box written “Automatically manage paging file size for all drives”

  7. Click on your hard drive if not selected

    Click on your hard drive then click customize size,then input the amount of size you want to add to your Virtual RAM (do not exceed your existing hard drive size) then click set and then apply

  8. Restart Your PC

    Restart your pc for changes to take effect and go back to see if it was successful.Enjoy
